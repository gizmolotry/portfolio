# Andrew Clair Portfolio

Single-page portfolio site for GitHub Pages.

## Local files

- `index.html` is the site.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder.
3. In the repository settings, enable GitHub Pages and set the source to the main branch root.
4. Your site will publish at the Pages URL GitHub gives you.
